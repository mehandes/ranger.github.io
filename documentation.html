Makefile

There are several ways the Makefile can help you to obtain documentation, for example by generating pydoc html pages. List all options by running "make options"

make: Test and compile ranger. make install: Install ranger make clean: Remove the compiled files (*.pyc, *.pyo) make doc: Create the pydoc documentation make cleandoc: Remove the pydoc documentation make man: Compile the manpage with "pod2man" make manhtml: Compile the html manpage with "pod2html" make snapshot: Create a tar.gz of the current git revision make test: Test all testable modules of ranger make todo: Look for TODO and XXX markers in the source code Options: PYTHON = /usr/bin/python PYOPTIMIZE = 1 DOCDIR = doc/pydoc DESTDIR = /
